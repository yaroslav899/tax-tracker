Run the following command:

```sh
npx create-turbo@latest
```

## What's inside?

- Monorepo (Turborepo)
- apps/web — React + Tailwind + Supabase Auth
- apps/api — Node.js + Express + TypeScript + Auth middleware
- PostgreSQL on Supabase + Prisma migrations
- Login/registration + route protection
